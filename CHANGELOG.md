# Changelog

All notable changes to `tiktoken-for-laravel` will be documented in this file.

## v0.1.7 - 2026-04-20

### What's Changed

- Fix CI matrix installation by skipping composer scripts during dependency update in `run-tests`.
- Improve cross-platform dependency constraints in workflow to avoid Windows shell constraint parsing issues.

## v0.1.6 - 2026-04-20

### What's Changed

- Add support for Laravel 13.
- Expand compatibility constraints for PHP and development dependencies.

## v0.1.5 - 2026-04-15

### What's Changed

- Bump `yethee/tiktoken` dependency to `^1.1.1`.
- Includes merged updates from recently merged PRs on `main`.

## 0.1.4 - 2025-03-05

Add support for Laravel 12.
Upgrade yethee/tiktoken to 0.7.0 to support o200k_base encoding.

## 0.1.3 - 2024-03-26

add support for Laravel 11
